INSTRUCCIONES RÁPIDAS

Tu página ahora está separada en 4 archivos:

1. index.html: estructura principal. Al principio no necesitas editarlo.
2. styles.css: colores y diseño visual. Cambia colores en :root.
3. products.js: archivo principal para editar nombre, WhatsApp, categorías y productos.
4. script.js: hace funcionar filtros, búsqueda y páginas de producto. No lo toques al principio.

CÓMO SUBIRLO A GITHUB
1. Entra a tu repositorio tienda-prueba.
2. Sube estos archivos: index.html, styles.css, products.js, script.js.
3. Si GitHub pregunta si quieres reemplazar index.html, acepta.
4. Espera unos minutos y abre tu página.

CÓMO ENTRAR A UNA PRENDA
Haz clic sobre una tarjeta de producto. La URL cambiará a algo como:
https://romanaxol.github.io/tienda-prueba/#producto/polera-oversize-negra

CÓMO AGREGAR UNA FOTO
1. Crea una carpeta img en GitHub.
2. Sube una foto, por ejemplo polera-negra.jpg.
3. En products.js cambia image: "" por image: "img/polera-negra.jpg".
4. Para galería usa gallery: ["img/polera-1.jpg", "img/polera-2.jpg"].
